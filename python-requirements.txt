# No Python dependencies required - netcheck is a pure Bash script
# 
# System dependencies (installed via apt/yum/pacman):
# - bash (>= 4.0)
# - telnet or netcat-openbsd
# - curl (for HTTP status checks)
# - openssl (for SSL certificate validation)
# - bc (for time calculations)
# - iproute2 or net-tools (for network interface display)
# - dnsutils (for DNS lookups)
# - iputils-ping (for ICMP ping tests)
#
# All dependencies are automatically installed by:
# - install.sh (system-wide installation)
# - build-deb.sh (DEB package)
# - build-snap.sh (Snap package)
